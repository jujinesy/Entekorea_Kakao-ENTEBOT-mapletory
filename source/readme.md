# 개요
- PC 메이플스토리의 각종 정보를 검색할 수 있는 카카오톡 봇 소스입니다.

# 상세
- maple-cody.js : 캐릭터의 코디 정보를 출력합니다.
- maple-dojang.js : 캐릭터의 무릉 최고기록, 시간을 출력합니다.
- maple-info.js : 캐릭터의 서버, 레벨, 직업, 인기도, 길드를 출력합니다.
- maple-union.js : 캐릭터의 유니온 레벨, 전투력, 일일코인 수급량을 출력합니다.
